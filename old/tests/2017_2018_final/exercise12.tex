                          \frac{1}{\theta}x^{-1 + \frac{1}{\theta}} & \text{при } x \in (0;\,1),  \\
                          0 & \text{при }x \not\in (0;\,1). \\
                        \end{cases}
\] Оценка параметра \(\theta\), найденная с помощью метода максимального
правдоподобия, равна
\begin{answerlist}
  \item \(X_1\)
  \item \(-\ln X_1\)
  \item \(\ln X_1\)
  \item \(-X_1\)
  \item \(\frac{1}{\ln X_1}\)
\end{answerlist}
\end{question}
