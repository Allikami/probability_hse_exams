
\begin{question}

распределения \(Bi(5, p)\). При каком значении константы \(c\) оценка
\(\hat{p} = c \bar{X}\) является несмещённой?
\begin{answerlist}
  \item \(\frac{1}{5}\)
  \item \(5\)
  \item \(\frac{1}{n}\)
  \item \(n\)
  \item \(1\)
\end{answerlist}
\end{question}

\begin{solution}
\begin{answerlist}
  \item Неверно
  \item Неверно
  \item Отлично
  \item Неверно
  \item Неверно
\end{answerlist}
\end{solution}
