
\begin{question}

на \((0, \theta)\) распределения. При каком значении константы \(c\)
оценка \(\hat{\theta} = c \bar{X}\) является несмещённой?
\begin{answerlist}
  \item \(2\)
  \item \(1\)
  \item \(\frac{1}{2}\)
  \item \(n\)
  \item \(\frac{1}{n}\)
\end{answerlist}
\end{question}

\begin{solution}
\begin{answerlist}
  \item Отлично
  \item Неверно
  \item Неверно
  \item Неверно
  \item Неверно
\end{answerlist}
\end{solution}
