
\begin{question}

\(\hat\mu = 3aX_1 + 4a^2X_2\) математического ожидания \(\mu = \E(X_i)\)
будет несмещённой при \(a\) равном
\begin{answerlist}
  \item \(1.2\)
  \item \(3\)
  \item \(-1\)
  \item \(0\)
  \item \(-3\)
\end{answerlist}
\end{question}

\begin{solution}
\begin{answerlist}
  \item Не угадал
  \item Тоже ересь
  \item Отлично
  \item Неверно
  \item Неверно
\end{answerlist}
\end{solution}
